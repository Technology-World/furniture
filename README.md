# 🛋️ Furniture Store

A multi-page frontend for a furniture e-commerce store. Includes Home, About, Shop, and Contact pages with a clean and modern UI, featuring an interactive homepage slider built with Swiper.js.

## 🔗 Live site

[furniture-app-umber.vercel.app](https://furniture-app-umber.vercel.app/)

## 📌 Features

- Multi-page layout with React Router
- Home, About, Shop and Contact pages
- Interactive homepage image slider using Swiper.js
- Product listings with clean UI
- Fully responsive design

---

## 🚀 Run locally

```bash
git clone https://github.com/Technology-World/furniture.git
cd furniture
npm install
npm run dev
```
